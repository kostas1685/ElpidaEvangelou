GitHub Ρυθμίσεις

Για να δημοσιεύετε άρθρα απευθείας στο live site, χρειάζεστε ένα GitHub Personal Access Token με δικαίωμα repo.

Πώς να το δημιουργήσετε:
1. Πηγαίνετε στο GitHub → Settings → Developer settings → Personal access tokens
2. Κάντε κλικ "Generate new token (classic)"
3. Επιλέξτε scope: repo
4. Αντιγράψτε το token και επικολλήστε το παρακάτω

Στοιχεία Εισόδου

Η πρόσβαση στο Studio προστατεύεται από τον server (Vercel). Για αλλαγή username/κωδικού, ενημερώστε τις μεταβλητές STUDIO_USER / STUDIO_PASS στο Vercel → Settings → Environment Variables.
